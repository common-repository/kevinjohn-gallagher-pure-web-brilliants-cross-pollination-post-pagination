Transforms the standard next/previous links to work across your choice of post types.


== Description ==

Presents you with a checkbox for each post type (separated by built-in and custom) to allow the NEXT / PREV link to be extended outwith the 1 to 1 relationship that exists within post types in WordPress.




== Installation ==

1. Upload `kevinjohn_gallagher___cross_pollination_post_pagination.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
